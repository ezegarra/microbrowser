How do I change XML indentation in IntelliJ IDEA? xml ide intellij-idea Sure there is. This is all you need to do: Go to File -> Settings -> Global Code Style -> General Disable the checkbox next to 'Use same settings for all file types' The 'XML' tab should become enabled. Click it and set the 'tab' (and probably 'indent') size to 2. In IntelliJ IDEA 10.0.3 it's File > Settings > Code Style > General
